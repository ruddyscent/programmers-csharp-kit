# Programmers C# Coding Test Kit

프로그래머스 스쿨의 [코딩테스트 고득점 Kit](https://school.programmers.co.kr/learn/challenges?tab=algorithm_practice_kit)를 C# 중심으로 학습하기 위한 풀이 노트 저장소입니다.

이 저장소는 문제를 그대로 모아 둔 문제집이 아니라, 각 유형별 핵심 아이디어와 C# 풀이 과정을 정리한 개인 학습 자료입니다.

## 코딩테스트 고득점 Kit

프로그래머스의 코딩테스트 고득점 Kit는 프로그래머스 팀이 코딩테스트 결과를 분석해, 코딩테스트에 자주 나오는 유형과 사람들이 많이 틀리는 유형을 간추린 연습 세트입니다.

공식 Kit에는 다음 유형이 포함되어 있습니다.

| 유형 | 공식 문제 수 | 이 저장소 |
| --- | ---: | --- |
| 해시 | 5 | 완료 |
| 스택/큐 | 6 | 완료 |
| 힙(Heap) | 3 | 완료 |
| 정렬 | 3 | 완료 |
| 완전탐색 | 7 | 완료 |
| 탐욕법(Greedy) | 6 | 예정 |
| 동적계획법(Dynamic Programming) | 5 | 예정 |
| 깊이/너비 우선 탐색(DFS/BFS) | 7 | 예정 |
| 이분탐색 | 2 | 예정 |
| 그래프 | 3 | 예정 |

## 자료 구성 방식

이 저장소에서 제시하는 풀이는 프로그래머스 코딩테스트 환경과 호환되는 C# 7.0을 기준으로 작성합니다.

각 문제는 보통 두 가지 관점의 풀이를 함께 다룹니다.

1. 이해하기 쉬운 버전
2. 짧고 간결한 버전

두 풀이 모두 가능한 범위에서 시간 복잡도와 공간 복잡도를 함께 분석합니다.

각 학습자료는 보통 다음 흐름으로 구성됩니다.

1. 문제 핵심 요약
2. 예제 이해
3. 풀이 아이디어
4. C# 코드
5. 시간 복잡도와 주의할 점

문제 원문과 전체 조건은 반드시 프로그래머스 공식 문제 페이지에서 확인하세요.

## 저작권 안내

프로그래머스 문제, 문제 설명, 제한사항, 입출력 예, 테스트케이스 등 원문 콘텐츠의 저작권은 프로그래머스 및 각 원저작자에게 있습니다.

프로그래머스 공식 사이트는 사이트의 콘텐츠, 정보, UI, HTML 소스 등에 대한 무단 복제, 전송, 배포, 크롤링, 스크래핑 등의 행위를 금지한다고 고지하고 있습니다.

따라서 이 저장소의 라이선스는 작성자가 만든 풀이, 해설, 코드, 학습 정리에만 적용됩니다. 프로그래머스의 문제 원문과 공식 콘텐츠에는 이 저장소의 라이선스가 적용되지 않습니다.

각 문제를 학습하거나 제출하려면 아래 공식 페이지를 이용하세요.

- [프로그래머스 코딩테스트 고득점 Kit](https://school.programmers.co.kr/learn/challenges?tab=algorithm_practice_kit)
- [프로그래머스 코딩테스트 연습](https://school.programmers.co.kr/learn/challenges)
