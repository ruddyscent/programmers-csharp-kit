# 🐾 폰켓몬 — C# 학습자료

## 📌 1. 문제 핵심

총 `N`마리의 폰켓몬이 있습니다.

우리는 그중에서 정확히 `N / 2`마리를 골라야 합니다.

목표는 다음과 같습니다.

```text
고른 폰켓몬들의 종류 수를 최대한 크게 만들기
```

예를 들어:

```text
nums = [3, 1, 2, 3]
```

폰켓몬은 총 4마리입니다.

따라서 고를 수 있는 마릿수는:

```text
4 / 2 = 2마리
```

전체 폰켓몬 종류는 다음과 같습니다.

```text
3번, 1번, 2번 → 3종류
```

하지만 우리는 2마리만 고를 수 있습니다.

2마리로 만들 수 있는 종류 수는 최대 2종류입니다.

따라서 정답은:

```text
2
```

입니다. ✅

---

## 🧠 핵심 아이디어

이 문제는 모든 조합을 직접 만들어 볼 필요가 없습니다.

예를 들어 `N = 10`이면 `5`마리를 골라야 합니다.

그런데 폰켓몬 종류가 아무리 많아도, 5마리만 고르면 종류 수는 최대 5종류입니다.

즉:

```text
최대로 고를 수 있는 종류 수 ≤ N / 2
```

입니다.

또한 전체 폰켓몬 종류 수보다 더 많은 종류를 고를 수도 없습니다.

```text
최대로 고를 수 있는 종류 수 ≤ 전체 종류 수
```

따라서 정답은 다음과 같습니다.

```text
min(전체 종류 수, N / 2)
```

🎯 이것이 이 문제의 핵심입니다.

# 🧠 풀이 1. 이해하기 쉬운 방법 — HashSet으로 종류 세기

## 💡 아이디어

`HashSet<int>`은 중복을 허용하지 않는 자료구조입니다.

예를 들어:

```text
[3, 1, 2, 3]
```

을 HashSet에 넣으면:

```text
3, 1, 2
```

만 남습니다.

즉, HashSet의 개수가 바로 폰켓몬의 종류 수입니다. 🧺

그다음:

```text
전체 종류 수
고를 수 있는 마릿수 N / 2
```

둘 중 더 작은 값을 반환하면 됩니다.

---

## 💻 C# 코드

```csharp
using System;
using System.Collections.Generic;

public class Solution
{
    public int solution(int[] nums)
    {
        HashSet<int> kinds = new HashSet<int>();

        foreach (int num in nums)
        {
            kinds.Add(num);
        }

        int maxPick = nums.Length / 2;

        if (kinds.Count > maxPick)
        {
            return maxPick;
        }

        return kinds.Count;
    }
}
```

---

## 🔎 코드 해설

### 1. HashSet 만들기

```csharp
HashSet<int> kinds = new HashSet<int>();
```

폰켓몬 종류 번호를 저장할 공간입니다.

HashSet은 같은 값을 여러 번 넣어도 한 번만 저장합니다.

```text
3, 3, 3을 넣어도 3은 하나만 저장
```

---

### 2. 폰켓몬 종류 넣기

```csharp
foreach (int num in nums)
{
    kinds.Add(num);
}
```

모든 폰켓몬 번호를 HashSet에 넣습니다.

예를 들어:

```text
[3, 3, 3, 2, 2, 4]
```

라면 HashSet에는 다음만 남습니다.

```text
3, 2, 4
```

따라서 종류 수는 3입니다. ✅

---

### 3. 고를 수 있는 마릿수 계산

```csharp
int maxPick = nums.Length / 2;
```

문제에서 반드시 `N / 2`마리를 골라야 한다고 했습니다.

---

### 4. 더 작은 값 반환

```csharp
if (kinds.Count > maxPick)
{
    return maxPick;
}

return kinds.Count;
```

종류가 아무리 많아도 고를 수 있는 마릿수보다 많은 종류를 고를 수는 없습니다.

그래서 더 작은 값을 반환합니다. 🎯

---

## ⏱️ 시간 복잡도

`nums` 배열을 한 번 순회합니다.

```text
O(n)
```

`n`은 `nums.Length`입니다.

---

## 📦 공간 복잡도

HashSet에 폰켓몬 종류를 저장합니다.

```text
O(k)
```

`k`는 폰켓몬 종류 수입니다.

최악의 경우 모든 폰켓몬이 다른 종류라면:

```text
O(n)
```

입니다.

# 🚀 풀이 2. 코드가 짧은 방법 — LINQ의 Distinct 사용하기

## 💡 아이디어

C#에는 중복을 제거해 주는 `Distinct()`가 있습니다.

```csharp
nums.Distinct()
```

이렇게 하면 중복된 폰켓몬 번호가 제거됩니다.

그 후 `Count()`로 종류 수를 셀 수 있습니다.

---

## 💻 C# 코드

```csharp
using System;
using System.Linq;

public class Solution
{
    public int solution(int[] nums)
    {
        return Math.Min(nums.Distinct().Count(), nums.Length / 2);
    }
}
```

---

## ✨ 이 코드의 의미

```csharp
nums.Distinct().Count()
```

전체 폰켓몬 종류 수를 구합니다.

```csharp
nums.Length / 2
```

고를 수 있는 폰켓몬 마릿수를 구합니다.

```csharp
Math.Min(...)
```

둘 중 더 작은 값을 반환합니다.

즉, 아래 식이 그대로 코드가 된 것입니다.

```text
정답 = min(전체 종류 수, N / 2)
```

⚡ 매우 짧고 깔끔한 풀이입니다.

---

## ⚠️ 프로그래머스에서 실행할 때 주의

`Distinct()`와 `Count()`를 쓰려면 반드시 아래 네임스페이스가 필요합니다.

```csharp
using System.Linq;
```

프로그래머스 C# 제출 코드 맨 위에 넣어야 합니다. ✅

---

## ⏱️ 시간 복잡도

`Distinct()`가 전체 배열을 한 번 순회하며 중복을 제거합니다.

```text
O(n)
```

입니다.

---

## 📦 공간 복잡도

내부적으로 중복 제거를 위한 HashSet 계열 구조를 사용합니다.

```text
O(k)
```

최악의 경우:

```text
O(n)
```

입니다.

# ⚖️ 두 풀이 비교

| 풀이 | 핵심 방법 | 장점 | 시간 복잡도 | 공간 복잡도 |
|---|---|---|---:|---:|
| 풀이 1 | `HashSet` 직접 사용 | 원리가 잘 보인다 😊 | O(n) | O(n) |

이유는 다음과 같습니다.

```text
1. 중복 제거 원리를 직접 이해할 수 있습니다.
2. 종류 수를 세는 과정이 눈에 잘 보입니다.
3. 해시 자료구조의 기본 사용법을 익힐 수 있습니다.
```

이 문제는 복잡하게 조합을 만들 필요가 없습니다.

```text
고를 수 있는 종류 수는

```

